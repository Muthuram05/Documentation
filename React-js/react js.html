REACT FEATURES

  • JSX -> Javascript XML
  • Components -> Components are independent and reusable bits of code.
  • Virtual DOM(React uses the virtual DOM as a strategy to compute minimal DOM operations when re-rendering the UI.)
  • Simplicity -> ReactJS uses JSX file which makes the application simple and to code as well as understand.Component-based approach which makes the code reusable
  • Performance -> ReactJS is known to be a great performer. This feature makes it much better than other frameworks out there today. The reason behind this is that it manages a virtual DOM.

Using Create React App

If you've previously installed create-react-app globally via npm install -g create-react-app, we recommend you uninstall the package using npm uninstall -g create-react-app or yarn global remove create-react-app to ensure that npx always uses the latest version.

Requirements

Node version >= 8.10
NPM version >= 5.6

NPX -> Node Package eXecute(5.2)

npx create-react-app my-app --template typescript

If you don't want to set the environment variable, another option is to modify the scripts part of package.json from:

"start": "react-scripts start"
to Linux (tested on Ubuntu 14.04/16.04) and MacOS (tested by aswin-s on MacOS Sierra 10.12.4):
"start": "PORT=3006 react-scripts start"
or (may be) more general solution by IsaacPak
"start": "export PORT=3006 react-scripts start"
Windows JacobEnsor's solution
"start": "set PORT=3006 && react-scripts start"
